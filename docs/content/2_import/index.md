# Import
This guide details how to import your [preprocessed](../0_preprocess/index.md) photos into [CoralNet](https://coralnet.ucsd.edu).

## Login
1.  Go to [coralnet.ucsd.edu](https://coralnet.ucsd.edu) and sign in.
2.  Select the [`CAMBioMed` source](https://coralnet.ucsd.edu/source/7958/) to which you want to add the photos.

## Upload images
1.  Go to the _Upload_ page.
2.  Click the _Upload images_ button.

    !!! note ""
        Check the source point generation settings and source annotation area settings. They should be:

        - Point generation settings: `simple random, 100 points`
        - Annotation area settings: `X: 0-100%, Y: 0-100%`

3.  Click _Browse_ and select the photos to upload.

    !!! tip ""
        Select multiple photos at the same time.

4.  Leave _Name prefix_ empty.
5.  Click _Start upload_ to begin uploading.

Once uploading is complete, you can continue uploading more files.


## Upload metadata
1.  Go to the _Upload_ page.
2.  Click the _Upload metadata_ button.
3.  Click _Browse_ and select the metadata CSV file.
4.  Click _Save metadata_ to apply the metadata.
